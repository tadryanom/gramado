
    ILI - Interactive Logon Infrastructure.


    O propósito é realizar todas as dependências de autenticação 
    de usuário para início de uma sessão do garden.
    
    Os componentes do ILI devem se comunicar com servidores dentro 
    do gramado para realizar as configurações dentro do gramado e 
    dentro do próprio ili. 

    + authentification.
    + user session
    + logon
    + credential providers
    + network providers.
    + security
    + product selector. 
    + iliinit
    + startili
    + startgde

    ...
