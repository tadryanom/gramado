# uapi

    Libs for Gramado operating system.
    It's gonna run in ring3.
    All the libs here will be used only in the gramado.


    libasm/ - Assembly ring3 library. 
    libc/   - libc ring 3 library.
    libgui/ - api for GUI applications.
    libio/  - i/o lib usade by drivers and servers in ring 3.
